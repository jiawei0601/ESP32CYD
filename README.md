# ESP32-CYD 多功能儀表板 (ESP32-Cheap-Yellow-Display)

這是一個專為 ESP32-CYD (ESP32-2432S028) 設計的高質感多功能儀表板。專注於流暢的 UI 體驗與實用的即時數據更新，讓您的桌邊顯示器充滿活力。

## 🚀 主要功能

- **📊 金融行情頁面**：
  - **市場總覽**：即時監控 NASDAQ 指數及熱門美股 (NVDA, GOOGL) 與加密貨幣 (BTC)。
  - **台股指數**：即時顯示台灣加權指數 (TAIEX) 的最新點數與漲跌幅。
- **🌤️ Aura 氣象引擎**：精美的動態氣象圖示（晴、雲、雨），顯示氣溫、體感溫度與風速。
- **🍅 番茄鐘計時器**：專為提升效率設計，支援 25 分鐘工作與 5 分鐘休息模式，具備動態進度顯示與按鈕回饋。
- **⚙️ 智慧設定系統**：內建 WiFi 掃描器與螢幕鍵盤，支援斷線自動重連機制。

## 🛠 硬體需求

- **ESP32-CYD (ESP32-2432S028)** 顯示器（內建 2.8 吋觸控螢幕）。
- **Micro USB 線**：用於供電與更新程式。

## ⚙️ 快速安裝

### 方法 A：直接燒錄 (推薦 🌟)
無需開發環境，僅需 1 分鐘即可完成：
1. 下載專案中的 `firmware_bundle` 資料夾。
2. 使用 [ESP32 網頁燒錄器 (Web Flasher)](https://web.esphome.io/)。
3. 依據以下位址填入檔案：
   - `bootloader.bin` -> **0x1000**
   - `partitions.bin`  -> **0x8000**
   - `firmware.bin`    -> **0x10000**

### 方法 B：原始碼編譯
1. 開啟 VS Code 的 **PlatformIO**。
2. 匯入本專案。
3. PlatformIO 將自動下載依賴庫 (`TFT_eSPI`, `ArduinoJson`, `XPT2046_Touchscreen`)。
4. 點擊 `Upload` 進行燒錄。

## 📱 操作指南

- **頂部導航**：點擊 Y 座標 0-24 範圍內的數字 (1-5) 快速切換功能。
- **頁面 1 (Market)**：全球金融走勢，採用分段異步加載，切換不卡頓。
- **頁面 2 (Stock)**：台灣股市即時焦點。
- **頁面 3 (Weather)**：氣象資訊，預設為新竹，可於設定中微調。
- **頁面 4 (Pomodoro)**：專注力工具。點擊 **START** 開始，**RESET** 歸零，結束後自動切換休息/工作模式。
- **頁面 5 (Settings)**：管理 WiFi 連線狀態。

## 📝 開發細節 (Development Notes)

- **非阻塞式設計**：數據抓取與介面渲染完全分離，確保網路連線不穩時，觸控響應依然靈敏。
- **介面保護機制**：採用局部 `fillRect` 刷新，徹底解決換頁時頂部導航欄閃爍或被覆蓋的問題。
- **數據可靠性**：整合 Brapi、Yahoo Finance 與 Open-Meteo 多重 API 路徑。
